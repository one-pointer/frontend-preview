01

앱에서 계정 삭제하는 방법 (권장)

쪽집게 앱에 로그인한 상태에서 아래 순서대로 진행하세요. 가장 빠르고 확실한 방법입니다.

1
앱 실행 후 마이페이지로 이동
쪽집게 앱을 열고 하단 탭바에서 "마이페이지" 탭을 눌러주세요.
2
"회원탈퇴" 메뉴 선택
마이페이지 하단 메뉴 목록에서 "회원탈퇴" 항목을 찾아 눌러주세요.
3
탈퇴 사유 선택
서비스 개선을 위해 탈퇴 사유를 선택해주세요. (필수 아님 — 더 나은 서비스를 만들기 위한 참고용입니다.)
4
안내 사항 확인 후 탈퇴 완료
삭제되는 정보에 대한 안내를 확인한 뒤 "탈퇴하기" 버튼을 누르면 즉시 처리됩니다.
⚠️
주의: 탈퇴 후에는 계정 복구가 불가능합니다. 진행 전에 꼭 아래 "탈퇴 전 확인사항"을 읽어주세요.
02

이메일로 계정 삭제 요청하기

앱에 로그인할 수 없거나 이미 앱을 삭제한 경우, 아래 이메일로 계정 삭제를 요청하실 수 있습니다.

📧 계정 삭제 요청 이메일
아래 이메일 주소로 다음 정보를 포함하여 요청해주시면, 본인 확인 후 처리해드립니다.
  • 수신 privacy@onepointer.co.kr
  • 제목 [계정 삭제 요청] 본인 이름
  • 본문 내용 • 가입 시 사용한 이메일 주소
    • 로그인 방식 (Google / Apple / Kakao)
    • 앱 내 닉네임 (기억하는 경우)
ℹ️
본인 확인 절차: 수신된 이메일의 발신자 주소와 가입 시 이메일이 일치하는지 확인합니다. 불일치 시 추가 본인 확인 절차가 진행될 수 있습니다.
03

삭제되는 데이터 / 보관되는 데이터

즉시 영구 삭제되는 데이터

✕ 이메일 ✕ 닉네임 ✕ 프로필 사진 ✕ 소셜 로그인 연동 ✕ FCM 토큰 ✕ 위치 정보 ✕ 업로드 사진·파일 ✕ 채팅 메시지 ✕ 작성한 리뷰 ✕ 전문가 프로필 ✕ 자격증·포트폴리오 ✕ 계좌 정보

법령에 따라 일정 기간 보관되는 데이터

전자상거래법 등 관련 법령에 따라 아래 정보는 보관 기간 동안 별도 분리 보관된 후 자동 파기됩니다. 이 데이터는 본인 식별이 불가능한 형태로 암호화되어 보관됩니다.

항목 보관 기간 근거
계약·청약철회 기록 5년 전자상거래법 §6
결제·재화 공급 기록 5년 전자상거래법 §6
소비자 불만·분쟁 처리 기록 3년 전자상거래법 §6
서비스 접속 로그 3개월 통신비밀보호법
🔒
안심하세요: 보관되는 데이터는 본인 식별이 불가능한 상태로 암호화되며, 오직 법적 의무 이행 목적으로만 접근 가능합니다.
04

삭제 처리 기간

방법 처리 시점
앱 내 회원탈퇴 즉시 처리 (실시간)
이메일 요청 접수 후 최대 7영업일 이내 처리

처리 완료 시 등록된 이메일로 완료 안내가 발송됩니다.

05

탈퇴 전 확인사항

🚫
탈퇴하면 복구할 수 없습니다: 한 번 삭제된 계정과 데이터는 어떠한 경우에도 복구되지 않습니다.

탈퇴 진행 전에 아래 사항을 반드시 확인해주세요.

  • 진행 중인 거래가 있다면 먼저 완료하거나 환불 처리를 받으세요. 진행 중 거래가 있으면 탈퇴가 제한될 수 있습니다.
  • 보유 중인 쿠폰/포인트는 모두 소멸되며 환불되지 않습니다.
  • 전문가로 활동 중인 경우 등록한 프로필, 자격증, 포트폴리오, 후기가 모두 삭제됩니다.
  • 인앱결제 내역은 Google Play / App Store에 별도로 보관되며, 환불은 각 플랫폼 정책을 따릅니다.
  • 탈퇴 후 동일 이메일로 재가입은 가능하지만, 기존 데이터는 복구되지 않습니다.
06

문의처

계정 삭제 관련 문의가 있으시거나 삭제 후에도 도움이 필요하신 경우 언제든 연락해주세요.
