WHY 쪽집게?

전문가 프로필이
곧 경쟁력입니다

쪽집게는 일반 재능마켓과 다릅니다. 의뢰인이 원하는 작업을 등록하면 전문가가 직접 선택해서 제안하는 구조예요.

📋

포트폴리오 & 자격증

의뢰인은 제안서를 받으면 가장 먼저 전문가 프로필을 확인합니다. 포트폴리오와 자격증이 충실할수록 제안서 수락 확률이 높아집니다.

✍️

상세 소개 작성

경력, 전문 분야, 서비스 제공 방식을 구체적으로 작성해주세요. 신뢰도가 높아지고, 제안서가 채택될 가능성이 커집니다.

🔗

프로필 공유로 고객 유치

내 프로필 링크를 기존 고객에게 공유하면, 수수료 0%로 거래할 수 있어요. 신규 가입 고객은 웰컴 쿠폰으로 무료 이용 가능!

📈

매칭률 = 프로필 완성도

같은 제안이라도 프로필이 잘 갖춰진 전문가가 선택됩니다. 프로필을 꼼꼼히 채울수록 더 많은 의뢰인과 연결될 수 있어요.

STEP 01

전문가 프로필 등록

마이페이지에서 5단계로 간편하게 등록하세요.

01 마이페이지

마이페이지에서
전문가 등록 시작

하단 탭의 '마이페이지'에서 '전문가' 탭을 선택하면 전문가 등록 배너가 보여요.

  • ✓의뢰인 / 전문가 탭 자유롭게 전환
  • ✓'전문가 등록' 배너를 탭하면 등록 시작
  • ✓제안서 관리, 직접 받은 의뢰 등 메뉴 노출
02 기본 정보

5단계로
프로필 완성

한 줄 소개, 상세 소개, 경력, 활동 방식, 카테고리, 자격증, 포트폴리오, 활동 시간/지역, 정산 계좌를 입력합니다.

  • ✓한줄 소개 — 나를 표현하는 짧은 문장 (최대 100자)
  • ✓활동 방식 — 오프라인 / 온라인 / 온·오프라인
  • ✓전문 카테고리 — 최대 3개 선택
🏦

온라인 거래 시 정산 계좌 등록 필수

오프라인 전용이라면 계좌 없이도 활동 가능합니다.

03 포트폴리오

자격증 & 포트폴리오로
전문성 어필

자격증은 최대 5개, 포트폴리오는 최대 10개까지 등록할 수 있어요.

  • ✓자격증 — 자격증명 + 발급기관 (최대 5개)
  • ✓포트폴리오 — 설명 + 이미지 (포트폴리오당 최대 7장)
  • ✓작성 후 수정/삭제 자유롭게 가능
⭐

자격증과 포트폴리오를 추가하면, 매칭률이 올라가요!

의뢰인이 전문가를 선택할 때 가장 중요하게 봅니다.

  • ✓카카오톡으로 공유 — 채팅으로 바로 전송
  • ✓링크 복사 — 어디든 붙여넣기
  • ✓다른 앱 공유 — 인스타 DM, 문자, 이메일 등

STEP 02

의뢰 탐색 & 제안서 보내기

의뢰인이 등록한 의뢰를 탐색하고, 마음에 드는 의뢰에 제안서를 보내세요. 제안서 발송은 무료!

의뢰 피드

다양한 의뢰를
한눈에 탐색

카테고리, 지역, 유형(온/오프라인), 정렬(최신순, 예산 높은순, 마감임박순)으로 필터링하세요.

  • ✓각 의뢰의 예산 범위, 마감일(D-day), 제안 수 한눈에
  • ✓온라인/오프라인 유형 태그 구분
  • ✓카테고리별 세부 필터
제안서 작성

매력적인 제안서로
의뢰인을 사로잡으세요

의뢰 상세를 확인하고 '제안서 보내기' 버튼으로 제안서를 작성합니다.

  • ✓제안 금액 + 예상 작업 시간
  • ✓어필 메시지 — 나만의 강점을 어필 (최대 1,000자)
  • ✓가능 일시 — 날짜와 시간대 추가
제안서 관리

보낸 제안서를
한곳에서 관리

마이페이지에서 진행중/완료 탭으로 상태를 구분하며, 대기 중인 제안서는 철회도 가능합니다.

  • ✓대기중 → 선택됨 → 완료
  • ✓선택되면 1:1 채팅방 자동 생성
  • ✓대기 중 제안서 철회 가능
ℹ️

의뢰인이 수락하면 채팅방이 열립니다

채팅방에서 세부 일정, 금액, 범위 등을 협의하게 됩니다.

STEP 03 — 가장 중요!

채팅 워크플로우

매칭 후 합의서 → 결제 → 작업물 전달 → 승인 → 정산까지의 전체 흐름

1
매칭 완료
채팅방 생성
2

3

에스크로 보관
4
작업물 전달
파일 / 메모
5
승인 → 정산
영업일 6~8일

🏠 오프라인 의뢰

  • 서비스 방식대면 서비스
  • 합의서없음 (채팅 협의)
  • 결제당사자 간 직접
  • 작업물 전달없음
  • 완료48시간 자동 완료

💻 온라인 의뢰

  • 서비스 방식제품 납품
  • 합의서필수 (금액·마감·범위)
  • 결제에스크로 안전결제
  • 작업물 전달앱에서 전달 필수
  • 완료승인 시 (72시간 자동)
온라인 의뢰 상세

단계별 상세 가이드

1

매칭 완료 → 채팅방 자동 생성

의뢰인이 내 제안서를 수락하면 자동으로 1:1 채팅방이 생성됩니다.

  • 상단 스텝바로 현재 진행 상태 파악
  • 온라인 의뢰: 의뢰인이 합의서를 작성·제안
💡

의뢰인과 충분히 대화하세요

금액, 작업 범위, 마감일을 미리 협의하면 합의서가 한 번에 확정될 가능성이 높아요.

2

합의서 확인 (온라인 전용)

의뢰인이 합의서를 제안하면 채팅방에 시스템 메시지가 나타납니다.

  • 작업 마감일, 작업 범위, 수정 횟수 확인
  • 납품 형식, 최종 금액 확인
  • 수락 → 결제 배너 노출 / 거부 → 재협의
3

의뢰인 결제 (에스크로)

합의서 수락 후 의뢰인이 결제하면, "결제가 완료되었습니다" 메시지가 전송됩니다.

  • 결제 대금은 PG사(토스페이먼츠)가 에스크로로 보관
  • 전문가 화면: "작업물을 전달해주세요" 배너로 전환
🔒

에스크로 안전결제

의뢰인이 작업물을 승인해야 전문가에게 정산됩니다. 양쪽 모두 안전합니다.

4

작업물 전달 (가장 중요!)

채팅방 상단의 '작업물 전달' 버튼으로 결과물을 전달합니다.

  • 메모로 전달 — 파일 없이 메모로 작성
  • 파일 전달 — 사진 최대 10장, PDF 최대 10개
🚨

반드시 읽어주세요!

영상 등 사진/PDF 외 작업물은 의뢰인과 협의 후 이메일 등으로 전달하세요. 하지만 쪽집게 앱에서 '작업물 전달' 버튼은 반드시 눌러야 합니다. 그래야 정산이 진행됩니다. 72시간 미확인 시 자동 승인됩니다.

5

의뢰인 확인 → 승인 또는 수정

의뢰인이 작업물을 확인 후 승인하거나 수정을 요청합니다.

  • 승인 → 거래 완료, 에스크로 정산 진행
  • 수정 요청 → 사유 입력 필수, 수정 횟수 차감
  • 수정 횟수 소진 후 거절 시 분쟁 조정 진행
6

승인 완료 → 자동 정산

의뢰인이 승인하면 등록한 정산 계좌로 영업일 6~8일 내 자동 입금됩니다.

  • 예: 300,000원 → PG 수수료 10,500원 = 289,500원 수령
  • 수익관리 페이지에서 정산 상태 실시간 확인
💰

중개 수수료 0%

쪽집게는 중개 수수료를 받지 않습니다. PG사 수수료(약 3.5%)만 차감됩니다.

  • ✓총 순수익 — 수수료를 뺀 실제 수익
  • ✓정산 완료 / 대기 상태 구분
  • ✓PG사 수수료 (약 3.5%) 합계 확인
  • ✓월별/주간/일별 차트로 추이 확인
