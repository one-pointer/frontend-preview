EXPERT ONBOARDING GUIDE

쪽집게 전문가
온보딩 가이드

쪽집게에서 전문가로 활동하기 위해 알아야 할 모든 것을 정리했습니다.
프로필 등록부터 수익 정산까지, 단계별로 안내합니다.

STEP 1

전문가 프로필 등록

마이페이지에서 5단계로 간편하게 전문가 프로필을 등록하세요.

쪽집게는 전문가 프로필이 곧 경쟁력입니다

쪽집게는 일반 재능마켓과 다릅니다.
전문가가 상품을 올리는 것이 아니라, 의뢰인이 원하는 작업을 등록하면 전문가가 직접 선택해서 제안하는 구조예요.

1

포트폴리오 & 자격증

의뢰인은 제안서를 받으면 가장 먼저 전문가 프로필을 확인합니다. 포트폴리오와 자격증이 충실할수록 전문성을 인정받아 제안서 수락 확률이 높아집니다.

2

상세 소개 작성

경력, 전문 분야, 서비스 제공 방식을 구체적으로 작성해주세요. 의뢰인이 더 많은 정보를 얻을 수 있어 신뢰도가 높아지고, 제안서가 채택될 가능성이 커집니다.

3

프로필 공유로 고객 유치

타 플랫폼에서 높은 수수료를 내고 있나요? 내 프로필 링크를 기존 고객에게 공유하면, 쪽집게에서 수수료 0%로 거래할 수 있어요. 신규 가입 고객은 웰컴 쿠폰으로 무료 이용 가능해서 전환이 쉽습니다!

4

매칭률 = 프로필 완성도

같은 제안이라도 프로필이 잘 갖춰진 전문가가 선택됩니다. 프로필을 꼼꼼히 채울수록 더 많은 의뢰인과 연결될 수 있어요.

1

마이페이지 → 전문가 등록

하단 탭의 '마이페이지'로 이동해서 '전문가' 탭을 선택하면, 전문가 등록 배너가 보여요. 배너를 탭하면 5단계 등록이 시작됩니다.

마이페이지 화면

  • 의뢰인 / 전문가 탭 전환 가능
  • '전문가 등록' 배너를 탭하면 등록 시작
  • 제안서 관리, 직접 받은 의뢰 등 전문가 메뉴 노출
2

기본 정보 입력 (1~5단계)

한 줄 소개, 상세 소개, 경력, 활동 방식(온라인/오프라인/둘 다), 전문 카테고리(최대 3개), 자격증, 포트폴리오, 활동 가능 시간과 지역, 정산 계좌를 입력합니다.

등록 폼 (5단계)

  • 한줄 소개 — 나를 표현하는 짧은 문장 (최대 100자)
  • 상세 소개 — 경험, 전문 분야, 강점 등을 자유롭게
  • 경력 — 예: "5년", "10년 이상"
  • 활동 방식 — 오프라인 / 온라인 / 온·오프라인
💡

TIP

외부 플랫폼(인스타, 페이스북), 전화번호, 이메일 등을 기재하면 제재 대상이 될 수 있습니다.

🏦

온라인 거래를 이용하려면 정산 계좌 등록이 필수입니다

5단계에서 정산 계좌를 등록해야 온라인 의뢰에 제안서를 보낼 수 있습니다. 오프라인 전용이라면 계좌 없이도 활동 가능합니다.

3

자격증 & 포트폴리오 관리

자격증은 최대 5개, 포트폴리오는 최대 10개까지 등록할 수 있어요. 포트폴리오에는 설명과 이미지를 함께 올려 어필하세요.

자격증 / 포트폴리오

  • 자격증 — 자격증명 + 발급기관 입력 (최대 5개)
  • 포트폴리오 — 설명 + 이미지 (포트폴리오당 최대 7장, 최대 10개)
  • 작성 후 수정/삭제 자유롭게 가능
⭐

자격증과 포트폴리오를 추가하면, 매칭률이 올라가요!

의뢰인이 전문가를 선택할 때 자격증과 포트폴리오를 중요하게 봅니다. 꼭 등록해주세요.

4

완성된 프로필

모든 단계를 완료하면 전문가 프로필이 완성됩니다. 이제 의뢰 피드에서 의뢰를 탐색하고 제안서를 보낼 수 있어요!

완성된 전문가 프로필

  • 소개, 포트폴리오, 자격증, 리뷰가 모두 노출
  • 리뷰 평점과 매칭 횟수가 표시되어 신뢰도 확보
  • 활동 시간, 지역, 본인인증 배지 등 노출
  • 프로필이 매력적일수록 의뢰인의 선택 확률이 높아져요
5

내 프로필 공유하기

완성된 프로필을 기존 고객이나 SNS에 공유하세요. 수수료가 비싼 타 플랫폼에서 받은 의뢰인에게 링크를 보내면, 쪽집게에서 수수료 0%로 거래할 수 있습니다.

프로필 공유 방법

  • 마이페이지 > 전문가 탭 > 내 전문가 프로필 진입
  • 화면 우측 상단의 공유 버튼을 탭
  • 카카오톡으로 공유 — 카카오톡 채팅으로 바로 전송
  • 링크 복사하기 — 링크를 복사하여 어디든 붙여넣기
  • 다른 앱으로 공유 — 인스타 DM, 문자, 이메일 등으로 공유
🎁

신규 가입 고객은 웰컴 쿠폰을 받아요!

내 프로필 링크를 통해 쪽집게에 가입한 의뢰인은 웰컴 쿠폰을 받아 무료로 의뢰를 등록할 수 있어요. 수수료 부담 없이 바로 거래를 시작할 수 있으니, 타 플랫폼 고객에게 적극적으로 공유해보세요!

STEP 2

의뢰 탐색 & 제안서 보내기

의뢰인이 등록한 의뢰를 탐색하고, 마음에 드는 의뢰에 제안서를 보내세요. 제안서 발송은 무료입니다!

1

의뢰 피드 탐색

홈 화면에서 카테고리별로 등록된 의뢰를 탐색하세요. 다양한 분야의 의뢰가 올라옵니다. 카테고리, 지역, 유형(온/오프라인), 정렬(최신순, 예산 높은순, 마감임박순)으로 필터링할 수 있어요.

의뢰 피드 화면

  • 의뢰 / 전문가 탭 전환
  • 카테고리별 세부 필터
  • 각 의뢰의 예산 범위, 마감일(D-day), 제안 수 한눈에 확인
  • 온라인/오프라인 유형 태그로 구분
2

의뢰 상세 확인 → 제안서 보내기

의뢰를 탭하면 상세 내용을 확인할 수 있어요. 내가 잘 할 수 있는 의뢰라면 '제안서 보내기' 버튼을 눌러 제안서를 작성합니다.

제안서 작성 항목

  • 제안 금액 — 서비스 대가 (필수)
  • 예상 작업 시간 — 분/시간/일/주/개월 단위 (협의 가능 체크 가능)
  • 서비스 제공 방식 — 온라인: 납품 방식 (링크, 이미지, PDF 등) / 오프라인: 장소 제안
  • 어필 메시지 — 의뢰인에게 나를 어필하는 메시지 (필수, 최대 1,000자)
  • 가능 일시 — 가능한 날짜와 시간대 추가
3

제안서 관리

보낸 제안서는 마이페이지 > 제안서 관리에서 확인할 수 있어요. '진행중'과 '완료' 탭으로 상태를 구분하며, 대기 중인 제안서는 철회도 가능합니다.

제안서 상태

  • 대기중 — 의뢰인이 아직 검토 중 (철회 가능)
  • 선택됨 — 의뢰인이 내 제안서를 수락! → 1:1 채팅방이 자동 생성
  • 완료 — 거래 완료된 제안서
  • 거절/철회 — 미채택되거나 내가 철회한 제안서
ℹ️

의뢰인이 제안서를 수락하면 1:1 채팅방이 열립니다

채팅방에서 세부 일정, 금액, 범위 등을 협의하게 됩니다.

STEP 3 — 가장 중요!

매칭 후 채팅방에서 협의 → 합의서 → 결제 → 작업물 전달 → 승인 → 정산까지의 전체 흐름을 단계별로 안내합니다.

1

매칭 완료 → 채팅방 자동 생성

의뢰인이 내 제안서를 수락하면 자동으로 1:1 채팅방이 생성됩니다. 채팅방 상단에는 현재 진행 상태가 스텝바로 표시되어 현재 어디까지 진행됐는지 한눈에 파악할 수 있어요.

채팅방 화면 (전문가 시점)

  • 상단 스텝바: 매칭완료 → 합의서 확정 → 결제 → 서비스 진행 → 작업물 전달
  • '합의서를 작성해주세요' 배너가 표시됨
  • 온라인 의뢰: 의뢰인이 합의서를 작성·제안합니다
💡

의뢰인과 충분히 대화하세요

의뢰인과 충분한 얘기를 나눈 뒤 합의서 작성을 부탁하세요. 금액, 작업 범위, 마감일 등을 채팅에서 미리 협의하면 합의서가 한 번에 확정될 가능성이 높아요.

2

합의서 확인 (온라인 의뢰 전용)

온라인 의뢰의 경우, 의뢰인이 합의서를 제안합니다. 채팅방에 "합의서가 제안되었습니다" 시스템 메시지가 나타나고, '합의서 확인하기' 버튼을 눌러 세부 내용을 확인하세요.

합의서 상세 확인 항목

  • 작업 마감일 — 예: 2026년 3월 30일 (월)
  • 작업 범위 — 예: 영상 제작까지 부탁드려요
  • 수정 횟수 — 기본 2회 (합의에 따라 변경 가능)
  • 납품 형식 — 예: 링크전송
  • 최종 금액 — 예: 300,000원

※ 위 내용은 예시이며, 실제 합의서 항목은 의뢰에 따라 달라집니다.

합의서 하단의 '수락' 또는 '거부' 버튼으로 응답합니다.

  • 수락 → 합의서 확정, 의뢰인에게 결제 배너 노출
  • 거부 → 채팅에서 재협의 후 의뢰인이 새 합의서 제안
3

의뢰인 결제 (에스크로)

전문가가 합의서를 수락하면, 의뢰인의 채팅 화면에 결제 배너가 나타납니다. 의뢰인이 합의서 금액을 에스크로로 결제하면, 전문가 채팅방에 "결제가 완료되었습니다" 시스템 메시지가 전송됩니다.

결제 흐름

  • 의뢰인 화면: "결제 대기중 300,000원" + 결제하기 버튼
  • 결제 완료 후: 금액이 에스크로에 보관됨 (바로 전문가에게 지급되지 않음)
  • 전문가 화면: "작업 진행중 — 작업물을 전달해주세요" 배너로 전환
🔒

에스크로 안전결제

결제 대금은 PG사(토스페이먼츠)가 보관합니다. 의뢰인이 작업물을 승인해야 전문가에게 정산됩니다. 양쪽 모두에게 안전한 구조입니다.

4

작업물 전달 (가장 중요!)

서비스가 완료되면 채팅방 상단의 '작업물 전달' 버튼을 눌러 작업물을 전달합니다. 전달 유형은 '메모로 전달'과 '파일 전달' 두 가지입니다.

작업물 전달 화면

  • 전달 유형 선택
  • • 메모로 전달 — 전달할 파일이 없는 경우, 메모에 내용을 작성
  • • 파일 전달 — 작업 결과물 파일을 첨부하여 전달 (사진 0/10, PDF 0/10)
  • 메모 — 의뢰인에게 전달할 메시지 작성 (작업 내용, 참고 사항 등)
  • 첨부파일 — 사진 최대 10장, PDF 최대 10개
🚨

반드시 읽어주세요 — 작업물 전달 필수 안내

이미지 or PDF 파일 외 작업물이라면, 의뢰인과 협의 → 개별 연락 후 작업물 전송하세요.

영상 전달은 서비스 지원 준비중입니다. 영상은 의뢰인과 협의 후 이메일 등으로 전송해주세요.

링크 등 사진/PDF 외 작업물은 메모에 작업물 링크를 작성하거나, 의뢰인과 상의하여 이메일 등으로 전달해주세요.

하지만, 쪽집게 어플에서 '작업물 전달' 버튼은 반드시 눌러야 합니다.
그래야 의뢰인이 작업물을 승인 후 결제가 정산되어요.

작업물 전달 후, 의뢰인이 72시간 내에 승인하지 않으면 자동 승인 처리됩니다.

5

의뢰인 확인 → 승인 또는 수정 요청

의뢰인이 작업물을 확인합니다. 승인하거나 수정을 요청할 수 있어요.

작업물 상세 화면 (의뢰인 시점)

  • 상태: 승인 대기중
  • 전달 유형, 수정 횟수(잔여), 작업 마감일 표시
  • 메모 내용 확인 + 첨부파일 다운로드
  • 72시간 미확인 시 자동 승인 안내 표시

의뢰인 선택지

  • 승인 → 거래 완료, 에스크로 정산 진행
  • 수정 요청 → 사유 입력 필수, 의뢰인이 새 마감일 직접 선택, 수정 횟수 차감
🔄

수정 요청 시

합의서에서 정한 수정 횟수(기본 2회, 합의에 따라 변경 가능)만큼 수정 요청이 가능합니다. 수정 요청 시 의뢰인이 새로운 마감일을 직접 선택하며, 전문가는 해당 마감일까지 수정 후 다시 전달합니다.

⚖️

수정 횟수 소진 후 의뢰인이 거절하면?

수정 횟수를 모두 사용한 뒤에도 의뢰인이 거절할 경우, 분쟁 조정에 들어갑니다. 전액 환불이 아니라 양측 사유를 검토하여 공정하게 조정합니다. 자세한 내용은 어플 설정 → 온라인 환불/분쟁 정책에서 확인하세요.

6

승인 완료 → 자동 정산

의뢰인이 작업물을 승인하면 거래가 완료됩니다. 채팅방에 "의뢰인이 작업물을 승인하였습니다. 거래가 완료되었습니다." 시스템 메시지가 전송되고, 등록한 정산 계좌로 영업일 기준 6~8일 내 자동 입금됩니다. (에스크로 PG사 정산 처리에 소요되는 기간입니다.)

정산 확인

  • 수익관리 페이지에서 정산 대기 → 정산 완료 상태 확인
  • 예시: 300,000원 → PG사 수수료 10,500원 = 289,500원 수령
💰

중개 수수료 0%

쪽집게는 중개 수수료를 받지 않습니다. PG사 결제 수수료(약 3.5%)만 차감되며, 나머지 전액을 전문가가 수령합니다.

⭐

리뷰는 언제 생성되나요?

온라인 의뢰: 의뢰인이 작업물을 승인(수락)하면 리뷰가 자동 생성됩니다.
오프라인 의뢰: 의뢰인이 '거래완료' 버튼을 누르면 리뷰가 자동 생성됩니다.
거래 완료 후 채팅 내역이 리뷰로 공개되며, 개인정보는 자동 마스킹됩니다. 공개하고 싶지 않은 메시지는 비공개 처리할 수 있어요.

오프라인 의뢰는 어떻게 진행되나요?

오프라인 의뢰는 합의서·에스크로·작업물 전달 단계가 없습니다. 매칭 후 채팅에서 일정·장소·금액을 협의하고, 약속된 시간에 만나서 대면 서비스를 진행합니다. 결제는 현금·이체 등 당사자 간 직접 진행하며, 서비스 완료 후 의뢰인이 완료 확인을 하거나 48시간 경과 시 자동 완료 처리됩니다.

수익 관리

마이페이지 > 수익관리에서 전체 수익 현황, 거래 내역, 정산 상태를 한눈에 확인하세요.

1

수익 대시보드

기간별(1개월/3개월/6개월/1년) 수익 현황을 확인할 수 있습니다. 총 순수익, 정산 완료, 정산 대기, PG사 수수료가 한눈에 표시됩니다.

수익 관리 화면

  • 총 순수익 — 전체 수익에서 수수료를 뺀 순수익
  • 정산 완료 — 이미 계좌에 입금된 금액
  • 정산 대기 — 아직 입금되지 않은 금액 (에스크로 PG사 정산 기준 영업일 6~8일 소요)
  • PG사 수수료 — 결제 수수료 합계 (약 3.5%)
  • 월별/주간/일별 차트로 수익 추이 확인
2

거래 내역 상세

각 거래건별로 의뢰명, 의뢰인, 완료 시점, 원래 금액, 수수료, 실수령액과 정산 상태를 확인할 수 있습니다.

거래 내역 예시

  • 거래별 원래 금액, PG사 수수료, 실수령액 확인
  • 정산 상태: 정산 대기 / 정산 완료

전체/정산완료/정산대기 필터로 조회 가능

3

리뷰 관리

거래 완료 후 채팅 내역이 자동으로 리뷰로 공개됩니다. 전문가 프로필의 신뢰도를 높여주는 핵심 자산이에요.

리뷰 관리 화면

  • 평균 별점과 총 리뷰 수, 매칭 완료 수 표시
  • 각 리뷰에는 의뢰인의 별점 + 실제 채팅 대화 스냅샷 포함
  • 평균 응답 시간, 첫 응답 시간도 표시 — 빠른 응답이 좋은 인상!
  • 전문가 답변도 리뷰에 남길 수 있음
🔒

개인정보 보호

개인정보(전화번호, 이메일 등)는 자동으로 마스킹됩니다. 거래 완료 후 24시간 내에 공개하고 싶지 않은 메시지를 직접 비공개 처리할 수도 있어요.
