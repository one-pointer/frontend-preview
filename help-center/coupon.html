쿠폰 · 결제 안내

쿠폰 구매부터 에스크로 결제, 정산까지 안내합니다.

1 쿠폰이란?

쪽집게는 전문가 수수료 0%.
다른 플랫폼의 숨은 수수료(10~30%) 대신, 의뢰인이 커피 한잔 가격의 쿠폰으로 의뢰를 등록합니다.

쿠폰은 의뢰 등록에 사용되는 이용권입니다. 전문가에게 수수료를 부과하는 대신, 의뢰 등록 시 쿠폰 1장이 소모되는 구조입니다.

3 쿠폰 사용 흐름

쿠폰 구매 → 의뢰 등록 (예약) → 매칭 확정 (소진) or 매칭 실패 (반환)

1. 쿠폰 구매

패키지를 선택해 쿠폰을 충전합니다.

2. 의뢰 등록 (예약)

의뢰를 등록하면 쿠폰 1장이 "예약" 상태가 됩니다. 아직 사용된 것이 아닙니다.

3. 매칭 확정 (소진)

전문가를 선택하면 쿠폰이 최종 소진됩니다. 이후 환불 불가.

4. 매칭 실패 (반환)

모집 기간(7일) 내 매칭이 안 되거나 직접 취소하면 쿠폰이 자동 반환됩니다.

상태설명전환 시점
사용 가능 (AVAILABLE)의뢰에 사용할 수 있는 상태구매 직후
예약됨 (RESERVED)의뢰 등록으로 묶인 상태의뢰 등록 시
소진됨 (CONSUMED)매칭 완료로 사용된 상태매칭 확정 시
반환됨 (RETURNED)매칭 실패로 돌아온 상태취소/만료 시

4 쿠폰 환불

구매 후 7일 이내, 모든 쿠폰이 미사용 상태인 경우에만 전액 환불이 가능합니다. 한 장이라도 의뢰에 사용된 이력이 있으면 환불할 수 없으며, 인앱결제(Apple/Google)는 각 스토어에서 별도로 환불을 신청해야 합니다.

5 에스크로 결제 (온라인 의뢰)

에스크로 안전결제
온라인 의뢰 시 결제 금액은 토스페이먼츠가 안전하게 보관합니다. 전문가에게 바로 전달되지 않으며, 작업물 승인이 완료된 후에 정산됩니다.
합의서 확정 → 에스크로 결제 → 작업 진행 → 작업물 승인 → 전문가 정산

1. 합의서 확정

의뢰인과 전문가가 금액, 마감일, 작업 범위를 합의합니다.

2. 에스크로 결제

합의 금액을 카드 또는 간편결제로 결제합니다. 전문가에게 바로 전달되지 않습니다.

3. 작업 진행

전문가가 합의된 범위 내에서 작업을 진행합니다.

4. 작업물 승인

의뢰인이 결과물을 확인하고 승인합니다.

5. 전문가 정산

의뢰인 승인 후 영업일 기준 6~7일 내 전문가에게 정산됩니다. (PG 수수료 3.5% + 송금 수수료 500원 차감)

쪽집게 중개 수수료는 0%입니다.
PG사 수수료 3.5%와 송금 수수료 500원을 제외한 금액이 전문가에게 지급됩니다.

예시) 합의 금액 100,000원 → PG 수수료 3,500원 → 송금 수수료 500원 → 전문가 정산 96,000원

7 자주 묻는 질문

쪽집게는 전문가에게 수수료를 부과하지 않는 대신, 의뢰인이 커피 한잔 가격의 쿠폰으로 의뢰를 등록하는 구조입니다. 다른 플랫폼의 숨은 수수료(10~30%)를 없애고, 전문가가 정당한 대가를 온전히 받을 수 있도록 하기 위함입니다.
모집 기간(7일) 내에 전문가를 선택하지 않으면 쿠폰이 자동으로 반환됩니다. 의뢰인이 직접 의뢰를 취소하는 경우에도 마찬가지입니다. 반환된 쿠폰은 다시 사용할 수 있습니다.
구매 후 7일 이내이고, 패키지 내 모든 쿠폰이 미사용(AVAILABLE) 상태일 때만 전액 환불이 가능합니다. 쿠폰을 하나라도 의뢰에 사용했다면 해당 패키지는 환불할 수 없습니다. 환불은 패키지 단위(전액)로만 처리됩니다.
인앱결제 건은 앱 내에서 환불할 수 없습니다. Apple App Store 또는 Google Play Store에서 직접 환불 요청해주세요. 스토어에서 환불이 승인되면 해당 쿠폰은 자동으로 회수됩니다.
의뢰를 등록하면 전문가들이 시간과 노력을 들여 제안서를 작성합니다. 매칭이 안 됐더라도 서비스가 이미 작동한 것이므로, 해당 쿠폰은 다른 의뢰에 재사용만 가능하고 현금 환불은 제한됩니다.
온라인 의뢰에서 합의서가 확정된 후 에스크로 결제를 진행합니다. 합의서에는 최종 금액, 마감일, 작업 범위 등이 포함되며, 전문가가 수락하면 결제 단계로 넘어갑니다.
아니요, 오프라인 의뢰는 에스크로 결제를 하지 않습니다. 오프라인 의뢰(대면 레슨)의 경우, 결제는 의뢰인과 전문가가 직접 협의하여 현금, 계좌이체 등 자유롭게 진행합니다.
의뢰인이 작업물을 승인한 후 영업일 기준 6~7일 이내에 전문가에게 정산됩니다. 정산 금액은 합의 금액에서 PG 수수료(3.5%)와 송금 수수료(500원)를 제외한 금액이며, 전문가가 등록한 계좌로 자동 입금됩니다.
