전문가 활동 가이드

등록부터 정산까지, 전문가 활동의 모든 것

1 전문가 등록

등록비, 제안서, 수수료 모두 0원. 쪽집게에서 전문가 활동은 완전 무료입니다.

전문가 등록은 별도 심사 없이 누구나 가능합니다.

항목필수 여부설명
전문 분야 (카테고리)필수대분류/중분류 선택 (최대 3개)
소개글필수나를 어필하는 소개
자격증/경력선택관련 자격증, 경력 사항
포트폴리오선택작업물/결과물 이미지
가능 시간대필수요일별 가능 시간
활동 지역선택오프라인 의뢰 매칭용
프로필 완성도가 높을수록 의뢰인에게 더 잘 노출됩니다. 소개글과 포트폴리오를 꼼꼼히 작성해 주세요.

2 의뢰 탐색

의뢰 피드에서 내 전문 분야와 일치하는 의뢰를 탐색합니다.

피드 탐색

카테고리, 의뢰 유형(오프라인/온라인), 지역 등으로 필터링

알림 수신

내 전문 분야와 일치하는 새로운 의뢰가 등록되면 푸시 알림

의뢰 상세 확인

의뢰 내용, 예산, 원하는 날짜, 장소 등 상세 정보 확인

관심 있는 의뢰를 발견하면 바로 제안서를 보내세요. 먼저 제안할수록 채택 확률이 높아집니다.

3 제안서 작성

의뢰인에게 자신을 어필하는 제안서를 보냅니다. 제안서 발송은 무료입니다.

항목필수설명
제안 금액필수서비스에 대한 희망 금액
가능 일시필수서비스 가능한 날짜와 시간대
어필 메시지필수의뢰인에게 보내는 어필/소개
의뢰당 제안서 수에 제한이 있습니다. 매칭 전까지 제안서 수정/취소가 가능합니다.

4 매칭 후 진행

의뢰인이 제안서를 수락하면 1:1 채팅방이 자동으로 생성됩니다.

오프라인 의뢰

채팅에서 일시, 장소, 금액 등을 협의한 뒤 직접 만나서 서비스를 진행합니다. 결제는 당사자 간 직접 진행합니다.

온라인 의뢰

채팅 협의

금액, 마감일, 작업 범위 등 세부사항 조율

합의서 확인

의뢰인이 제안한 합의서(금액/마감일/범위/수정횟수)를 확인하고 수락 또는 거절

에스크로 결제

의뢰인이 합의 금액 결제 (토스페이먼츠가 안전 보관)

서비스 진행

합의된 마감일까지 작업

작업물 전달

완료 보고 또는 결과물 파일 전달

의뢰인 승인

의뢰인이 확인 후 승인 → 정산 진행

5 정산

쪽집게 중개 수수료는 0%입니다. 합의 금액에서 PG 수수료 3.5%만 제외한 전액이 정산됩니다.
항목온라인 의뢰오프라인 의뢰
결제 방식에스크로 (토스페이먼츠 보관)당사자 간 직접
중개 수수료0%0%
PG 수수료3.5%없음
정산 시점작업물 승인 후 영업일 6~8일 (에스크로 PG사 정산)즉시 (직접 수령)
정산 방식등록 계좌 자동 입금직접 수령

예시) 합의 금액 50,000원 → PG 수수료 1,750원 → 정산 금액 48,250원

6 환불 요청 대응

의뢰인이 환불을 요청하면 알림을 받게 됩니다.

상황대응 방법기한
환불 요청 수신수락 또는 거절 선택48시간
수락 시즉시 전액 환불 처리—
거절 시자동으로 분쟁 조정 전환—
무응답 시48시간 후 자동 전액 환불—
48시간 내에 반드시 응답해 주세요. 무응답 시 자동으로 전액 환불 처리됩니다.

거절 시 자동으로 분쟁 조정이 시작됩니다. 관리자가 채팅 내역과 합의서를 검토하여 공정하게 판단합니다. 자세한 분쟁 처리 절차는 환불/분쟁 정책을 참고해 주세요.

7 자주 묻는 질문

네, 완전 무료입니다. 등록비, 제안서 비용, 중개 수수료 모두 0원입니다.
네, 여러 의뢰에 동시에 제안서를 보낼 수 있습니다.
매칭 전까지는 제안서를 수정하거나 취소할 수 있습니다.

직접적인 불이익은 없으나, 분쟁 조정에서 채팅 내역과 합의서를 기반으로 판단됩니다.
