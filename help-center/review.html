리뷰 시스템

별점이 아닌 실제 대화로 전문가를 판단합니다

1 채팅 = 리뷰

쪽집게의 리뷰는 다릅니다.
텍스트 리뷰를 직접 작성하는 것이 아니라, 실제 채팅 대화가 그대로 리뷰로 공개됩니다. 전문가의 소통 방식, 전문성, 태도를 있는 그대로 확인할 수 있습니다.

왜 채팅 기반 리뷰인가요?

기존 별점/텍스트 리뷰의 한계:

  • 허위 리뷰, 과장된 리뷰의 가능성
  • 별점만으로는 알 수 없는 소통 능력

채팅 리뷰의 장점:

  • 전문가의 실제 소통 방식 확인
  • 조작 불가능
  • 다른 의뢰인과의 실제 대화 패턴 파악 가능

2 리뷰 생성 과정

서비스 완료

의뢰인이 서비스 완료를 확인하거나 자동 완료 처리

채팅 스냅샷 생성

시스템이 채팅 내역을 자동으로 스냅샷으로 저장

비공개 처리 기간

의뢰인에게 비공개 처리할 메시지를 선택할 기회 제공 (48시간)

자동 마스킹

전화번호, 계좌번호 등 민감 정보가 자동으로 마스킹 처리

리뷰 공개

전문가 프로필에 리뷰가 공개됨

리뷰는 자동으로 생성됩니다. 별도로 작성할 필요가 없어 간편합니다.

3 비공개 처리

채팅 내용 중 공개를 원하지 않는 메시지가 있다면 비공개 처리할 수 있습니다.

항목내용
비공개 처리 가능 시점서비스 완료 후 48시간 이내
처리 방법채팅방에서 원하는 메시지를 선택하여 비공개 처리
비공개 처리 후해당 메시지는 리뷰에서 숨겨짐
취소 가능 여부비공개 처리 기간 내에는 취소 가능
비공개 처리 기간(48시간)이 지나면 변경할 수 없습니다. 신중하게 선택해 주세요.

4 자동 마스킹

리뷰에 공개되는 채팅 내역에서 민감한 개인정보는 자동으로 마스킹 처리됩니다.

마스킹 대상예시마스킹 결과
전화번호010-1234-5678010-****-5678
계좌번호110-123-456789110-***-******
이메일user@email.comu***@email.com
자동 마스킹은 시스템이 자동으로 처리합니다. 추가로 비공개 처리를 원하는 메시지는 직접 선택해 주세요.

5 별점

채팅 리뷰와 별도로 별점을 입력할 수 있습니다. 별점은 선택사항이며, 서비스 완료 후 1회만 입력 가능합니다.

항목내용
별점 범위1~5점 (0.5점 단위)
필수 여부선택사항
입력 시점서비스 완료 후
수정 가능입력 후 수정 불가

6 리뷰 확인하기

다른 의뢰인의 리뷰를 확인하여 전문가를 판단할 수 있습니다.

전문가 프로필 방문

전문가 프로필 페이지에서 리뷰 탭 확인

채팅 리뷰 확인

실제 대화 내역을 통해 전문가의 소통 방식 파악

별점 확인

평균 별점과 별점 분포 확인

의뢰 판단

리뷰를 참고하여 제안서 수락 여부 결정

7 자주 묻는 질문

아닙니다. 비공개 처리한 메시지는 리뷰에서 숨겨지고, 민감 정보는 자동 마스킹됩니다.
리뷰 자체를 삭제할 수는 없지만, 개별 메시지를 비공개 처리할 수 있습니다.
현재는 의뢰인의 채팅 리뷰만 공개되며, 전문가의 별도 답변 기능은 추후 제공 예정입니다.
별점은 선택사항이므로 입력하지 않아도 채팅 리뷰는 정상적으로 공개됩니다.
네, 오프라인 의뢰도 채팅 내역이 리뷰로 공개됩니다. 매칭 후 채팅 협의 내역이 대상입니다.
