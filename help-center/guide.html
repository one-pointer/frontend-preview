이용 가이드

의뢰인과 전문가, 각 역할별 이용 흐름을 안내합니다.

1 서비스 한눈에 보기

쪽집게는 중개 수수료 없는 전문가 매칭 플랫폼입니다. 다른 플랫폼의 숨은 수수료(10~30%)를 없애고, 전문가 수수료 0%로 합의 금액 전체가 전문가에게 돌아갑니다. 커피 한잔 가격의 쿠폰으로 의뢰를 시작하세요.
STEP 1

의뢰 등록

쿠폰 1장으로 의뢰를 등록

STEP 2

제안서 비교

여러 전문가의 제안서를 비교

STEP 3

매칭 · 채팅

전문가와 1:1 채팅으로 협의

STEP 4

서비스 완료

완료 확인 후 리뷰 자동 공개

2 단계별 상세 설명

쿠폰 준비

쿠폰 패키지를 구매합니다. 커피 한잔 가격의 쿠폰이 의뢰 등록에 필요합니다. 쿠폰은 앱 내에서 카드 또는 간편결제로 충전할 수 있습니다.

의뢰 등록

카테고리, 의뢰 유형(오프라인/온라인), 예산, 원하는 날짜를 작성하여 의뢰를 등록합니다. 등록 시 쿠폰 1장이 예약(RESERVED)되며, 모집 기간은 등록일로부터 7일입니다.

전문가 제안

관련 분야 전문가들이 의뢰를 확인하고, 금액/가능 일시/어필 메시지를 담은 제안서를 발송합니다. 제안서 발송은 무료이며, 여러 전문가의 제안서를 비교할 수 있습니다.

매칭 확정

마음에 드는 전문가의 제안서를 수락하면 매칭이 확정됩니다. 이때 쿠폰이 소진(CONSUMED)되고, 1:1 채팅방이 자동으로 생성됩니다. 미채택 제안서는 자동 반려됩니다.

채팅 협의

1:1 채팅에서 세부사항을 조율합니다. 오프라인 의뢰는 채팅 후 직접 만남으로 진행하고, 온라인 의뢰는 합의서(금액/마감일/범위/수정횟수)를 작성한 뒤 에스크로 결제로 진행합니다.

서비스 완료

서비스가 완료되면 완료 확인 후 채팅 내역이 리뷰로 자동 공개됩니다. 별점도 입력할 수 있습니다. 민감 정보는 자동 마스킹되며, 원하는 메시지를 비공개 처리할 수도 있습니다.

3 오프라인 vs 온라인 비교

쪽집게의 의뢰는 오프라인 과 온라인 두 가지 유형으로 나뉩니다. 유형에 따라 진행 방식이 다릅니다.

(현금, 이체 등 자유)		에스크로 안전결제
(합의서 금액 기준)
합의서없음

(금액/마감일/범위/수정횟수)

(승인/수정요청/거부 가능)

4 자주 묻는 질문

네, 한 계정으로 두 역할 모두 이용 가능합니다. 마이페이지에서 의뢰인/전문가 모드를 자유롭게 전환할 수 있습니다. 전문가로 활동하려면 전문가 프로필 등록이 필요합니다.
구분의뢰인전문가
의뢰 등록O (쿠폰 1장)O (전문가도 의뢰 등록 가능)
제안서 발송XO (무료)
프로필 등록XO (무료)
전환 방법마이페이지에서 의뢰인/전문가 탭 전환
의뢰 등록일로부터 7일입니다. 7일 이내에 전문가를 선택하지 않으면 의뢰가 자동 만료되고, 예약된 쿠폰은 반환됩니다.
매칭 전(모집 중/검토 중 상태)에 취소하면 예약된 쿠폰이 자동으로 반환됩니다. 매칭이 확정된 이후에는 쿠폰이 이미 소진되었으므로 환불이 불가합니다.
쪽집게에서는 별도의 텍스트 리뷰를 작성하지 않습니다. 거래 완료 후 실제 채팅 내역이 리뷰로 자동 공개됩니다. 전화번호, 계좌번호 등 민감 정보는 자동 마스킹 처리되며, 원하는 메시지를 비공개 처리할 수도 있습니다.
필요한 전문 서비스를 의뢰하고, 전문가의 제안서를 비교해 직접 선택하세요. 쿠폰 1장으로 의뢰가 시작됩니다.
1

의뢰 등록

오프라인 / 온라인

배우고 싶은 것을 의뢰로 등록합니다. 아래 정보를 작성하세요.

  • 카테고리 — 원하는 분야를 선택하세요
    • 디자인 — 로고/아이콘, 포토샵, 앱 디자인, 웹 디자인, 그래픽 디자인
    • 스포츠 — 골프, 테니스, PT, 필라테스
    • 영상 — 광고/홍보, 숏폼, 영상 편집
    • IT — 웹 제작, 앱 제작, 컴퓨터 조립, 퍼블리싱
  • 의뢰 유형 — 오프라인(대면) 또는 온라인(화상/원격)
  • 예산 — 희망 예산 범위 입력
  • 원하는 날짜 — 서비스를 받고 싶은 날짜/시간대
  • 상세 설명 — 어떤 도움이 필요한지 구체적으로 작성
등록 시 쿠폰 1장이 예약(RESERVED)됩니다. 아직 소진이 아니므로, 매칭 전 취소/만료 시 자동 반환됩니다. 모집 기간은 등록일로부터 7일입니다.
2

제안서 비교 및 전문가 선택

오프라인 / 온라인

전문가들이 보낸 제안서를 비교하고 마음에 드는 전문가를 선택합니다.

  • 금액 — 전문가가 제시하는 서비스 금액
  • 가능 일시 — 전문가가 가능한 날짜와 시간
  • 어필 메시지 — 전문가의 경력, 강점, 진행 방식 소개
제안서가 도착하면 푸시 알림으로 알려드립니다. 이메일 알림은 지원하지 않으며, 앱 푸시 알림만 발송됩니다.

전문가의 프로필, 자격증, 포트폴리오를 확인하고 실제 채팅 리뷰를 통해 전문가의 커뮤니케이션 스타일까지 파악한 뒤 선택하세요.

제안서 목록에서 전문가 프로필을 탭하면 포트폴리오와 채팅 리뷰를 확인할 수 있습니다.
3

채팅 협의

오프라인 / 온라인

제안서를 수락하면 1:1 채팅방이 자동으로 생성됩니다. 채팅에서 아래 사항을 전문가와 조율하세요.

  • 정확한 서비스 일시와 장소 (오프라인) 또는 화상 링크 (온라인)
  • 서비스 범위와 진행 방식
  • 최종 금액과 기타 조건
채팅이 곧 거래 기록입니다. 가격, 마감일, 작업 범위 등 중요한 합의 사항은 반드시 채팅에서 명확하게 기록해 두세요. 거래 완료 후 채팅 내역이 리뷰로 공개됩니다.
4

합의서 작성

온라인 전용

온라인 의뢰는 에스크로 결제 전에 합의서를 작성해야 합니다. 의뢰인만 합의서를 제안할 수 있으며, 전문가가 수락하면 확정됩니다.

항목설명
최종 금액채팅 협의 후 확정된 서비스 금액
마감일전문가가 결과물을 전달해야 하는 기한
작업 범위서비스에 포함되는 구체적인 내용
수정 횟수작업물 수정 요청 가능 횟수 (기본 2회)

전문가가 합의서를 거절하면 채팅에서 재협의한 뒤 새 합의서를 제안할 수 있습니다.

5

에스크로 결제

온라인 전용

합의서가 확정되면 에스크로 결제를 진행합니다. 결제 대금은 토스페이먼츠가 안전하게 보관하며, 서비스 완료 후 전문가에게 정산됩니다.

의뢰인 결제 → 토스페이먼츠 보관 → 작업 완료 확인 → 전문가 정산
  • 카드, 간편결제(카카오페이, 토스 등) 사용 가능
  • 중개 수수료 0% — 합의 금액 전체가 전문가에게 지급 (PG 3.5% 별도)
  • 결제 후 2시간 이내 무조건 환불 가능 (쿨링오프)
6

작업물 확인

온라인 전용

전문가가 전달한 결과물을 확인하고 아래 세 가지 중 하나를 선택합니다.

선택설명
승인작업물에 만족하면 승인합니다. 거래가 완료되고 전문가에게 정산됩니다.
수정 요청수정이 필요하면 사유를 입력하여 요청합니다. 수정 요청 시 마감일이 자동으로 3일 연장됩니다. 합의서에 정한 횟수(기본 2회)까지 가능합니다.
거부합의 내용과 현저히 다를 경우 거부할 수 있으며, 자동으로 분쟁 조정이 시작됩니다.
작업물 전달 후 72시간 이내에 응답하지 않으면 자동으로 승인 처리됩니다. 전문가에게 정산이 진행되므로 반드시 기한 내에 확인하세요.
7

완료 확인

오프라인 전용

대면 서비스가 끝나면 채팅방에서 완료 확인 버튼을 눌러 거래를 종료합니다.

  • 완료 확인 후 채팅 내역이 리뷰로 자동 공개됩니다.
  • 48시간 동안 완료 확인을 하지 않으면 자동으로 완료 처리됩니다.
채팅방 상단의 "완료 확인" 버튼을 탭하세요.
8

환불 요청

온라인 전용

온라인 에스크로 결제 후 환불이 필요한 경우, 채팅방에서 환불을 요청할 수 있습니다.

시점환불 조건
결제 후 2시간 이내즉시 전액 환불 (전문가 동의 불필요)
2시간 이후 ~ 작업물 전달 전전문가 동의 필요 (48시간 무응답 시 자동 환불)
작업물 전달 후환불 요청 불가 (수정 요청 또는 거부만 가능)

자세한 환불 정책은 온라인 거래 환불/분쟁 정책을 참고하세요.

등록비, 제안서, 수수료 모두 무료입니다. 의뢰 피드에서 기회를 찾고, 제안서를 보내 매칭을 시작하세요.
1

전문가 프로필 등록

완전 무료 · 별도 심사 없이 바로 활동

전문가로 활동하려면 프로필을 등록해야 합니다. 등록비는 완전 무료이며 별도 심사 없이 바로 활동할 수 있습니다.

  • 전문 분야 — 활동할 카테고리 선택 (복수 선택 가능)
  • 자기 소개 — 경력, 강점, 진행 방식 등 자유 작성
  • 자격증 — 관련 자격증 이미지 업로드 (선택)
  • 포트폴리오 — 이전 작업물 이미지/설명 업로드 (선택)
  • 가능 시간대 — 요일별 활동 가능한 시간 설정
  • 가능 지역 — 오프라인 의뢰를 위한 활동 지역 설정
  • 정산 계좌 — 온라인 거래 이용 시 필수 등록 (오프라인 전용이면 미등록 가능)
프로필 등록 즉시 공개됩니다. 별도의 관리자 인증 없이 바로 활동할 수 있습니다. 프로필 완성도가 높을수록 의뢰인에게 더 잘 노출됩니다.
※ 추후 관리자 검증 기능이 추가될 수 있습니다.
2

의뢰 탐색

피드에서 내 분야 의뢰를 찾으세요

의뢰 피드에서 내 전문 분야와 일치하는 의뢰를 탐색합니다.

  • 카테고리 필터 — 전문 분야별로 의뢰 검색
  • 유형 필터 — 오프라인/온라인 구분
  • 지역 필터 — 오프라인 의뢰의 활동 지역
  • 알림 — 내 분야에 새로운 의뢰가 등록되면 푸시 알림을 받을 수 있습니다
3

제안서 발송

발송 비용 무료

마음에 드는 의뢰에 제안서를 보냅니다. 제안서 발송은 완전 무료입니다.

  • 서비스 금액 — 의뢰인에게 제안하는 금액
  • 가능 일시 — 서비스 가능한 날짜와 시간
  • 어필 메시지 — 의뢰인을 설득하는 자기 어필

제안서는 매칭 전까지 수정하거나 취소할 수 있습니다. 의뢰당 제안서 수에 제한이 있으므로, 신중하게 작성하세요.

4

채팅 상담

오프라인 / 온라인

의뢰인이 제안을 수락하면 1:1 채팅방이 열립니다. 일시, 장소, 서비스 범위 등을 상세히 조율하세요.

채팅 내역은 거래 완료 후 리뷰로 공개됩니다. 친절하고 전문적인 상담은 향후 의뢰인들이 전문가를 선택하는 데 큰 영향을 미칩니다.
5

합의서 확인

온라인 전용

의뢰인이 제안한 합의서를 확인하고 수락 또는 거절합니다.

항목확인 사항
최종 금액채팅에서 합의한 금액과 일치하는지 확인
마감일결과물 전달이 가능한 기한인지 확인

6

작업물 전달

온라인 전용

합의서에 정한 마감일까지 결과물(완료 보고 또는 파일)을 의뢰인에게 전달합니다.

  • 의뢰인이 승인하면 거래 완료 → 정산이 진행됩니다.
  • 의뢰인이 수정 요청하면 마감일이 자동으로 +3일 연장됩니다. 수정 후 재전달하세요.
  • 의뢰인이 거부하면 자동으로 분쟁 조정이 시작됩니다.
  • 의뢰인이 72시간 동안 응답하지 않으면 자동 승인 처리됩니다.
마감일이 지나면 의뢰인이 마감일 연장 또는 환불 요청을 선택할 수 있습니다. 마감일을 지키는 것이 중요합니다.

환불 요청 응답

온라인 전용

의뢰인이 환불을 요청하면 알림이 발송됩니다. 48시간 이내에 수락 또는 거절해야 합니다.

선택결과
수락즉시 전액 환불이 진행됩니다.
거절자동으로 분쟁 조정이 시작됩니다. 관리자가 채팅 내역과 합의서를 검토하여 공정하게 판단합니다.
48시간 동안 응답하지 않으면 자동으로 전액 환불 처리됩니다. 환불 요청 알림을 받으면 반드시 기한 내에 응답하세요.

자세한 환불 및 분쟁 정책은 온라인 거래 환불/분쟁 정책을 참고하세요.
